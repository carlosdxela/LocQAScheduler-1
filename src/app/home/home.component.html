Select any of the options in the main menu or below

  • Testers

    The user can view, add and edit the list of Testers

  • Project and Tasks

    The user can view, add and edit the List of Projects, as well as viewing and editing the list of Tasks within a Project. Assignments for a task can also be updated.

  • Assignments

    The user can view a flat list of task-tester assignments and filter them by Tester.
